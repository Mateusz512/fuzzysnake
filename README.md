# Fuzzy Snake

## Prerequisites

1. Install `OpenJDK 11`<br>
   ```shell script
    sudo apt install openjdk-11-jdk
   ```

## Build & Run

1. Build<br>
    ```shell script
    ./gradlew clean build
    ```
2. Run<br>
    ```shell script
    ./gradlew run
    ```
   
Alternatively you can do all at once:
```shell script
./gradlew clean build run
```
